# Tech Compass

A simple web-based compass visualization tool.

## Project Structure

- `index.html` — Main web page for the compass visualization
- `index.js` — JavaScript logic for compass functionality
- `styles.css` — Stylesheet for the project
- `images/` — Image assets used in the project
- `test_compass_image.html` — Test page for compass image
- `test_image.html` — Test page for image rendering

## Features

- Interactive compass display
- Responsive design
- Image-based compass rendering

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your web browser to view the compass.

## Development

- Edit `index.js` to modify compass logic.
- Update `styles.css` for custom styles.
- Add or replace images in the `images/` folder as needed.
